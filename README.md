# Bocatto – App de búsqueda de restaurantes

Bocatto es una aplicación desarrollada como **proyecto académico grupal**, orientada a facilitar la búsqueda de restaurantes según preferencias alimentarias, ubicación y valoraciones de otros usuarios.

El proyecto fue desarrollado utilizando **Flutter y Firebase**, con foco en la experiencia de usuario, accesibilidad y centralización de información gastronómica.

---

## Información general

- **Nombre del proyecto:** Bocatto  
- **Tipo:** Proyecto académico grupal  
- **Plataforma:** Web
- **Ámbito:** Ciudad de Mendoza, Argentina  
- **Estado:** Funcional (entorno Firebase)

---

## Objetivo del proyecto

Facilitar la búsqueda de restaurantes en un solo lugar, ampliando el abanico de opciones gastronómicas para residentes y turistas, y promoviendo la competencia entre locales mediante reseñas y puntuaciones.

---

## Público objetivo

- Ciudadanos de Mendoza  
- Turistas que visitan la provincia  
- Personas con preferencias o restricciones alimentarias

---

## Funcionalidades principales

- Visualización de restaurantes cercanos mediante mapa
- Sistema de puntuación con estrellas
- Reseñas y comentarios de usuarios
- Guardar restaurantes como favoritos
- Autenticación de usuarios (login / registro)
- Integración con mapa (OpenStreetMap)

---

## Filtros disponibles

- Sin TACC  
- Vegetariano  
- Vegano  
- Sin lactosa  
- Saludable  
- Comida nacional  
- Comida internacional  
- Desayuno / Merienda  
- Bodegón  
- Bares  
- Comida rápida  
- Con reserva

## Tecnologías utilizadas

### Desarrollo
- **Lenguaje:** Dart  
- **Framework:** Flutter  

### Backend / Servicios
- **Firebase**
  - Firestore (base de datos)
  - Authentication (usuarios)
  - Hosting (web)
  - Storage

### UX/UI
- **Diseño:** Figma  
- Diseño de pantallas, flujos de usuario y prototipos

### APIs externas
- OpenStreetMap (mapas)

---

## Arquitectura (Firebase)

- Base de datos NoSQL modelada en **Firestore**
- Autenticación de usuarios con Firebase Auth
- Datos organizados por colecciones y documentos
- Acceso seguro mediante reglas de Firebase
